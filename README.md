# OVOS Poetry Skill

OVOS Skill that reads requested poetry.  

## About

The poetry skill will recite poems from a pre-loaded dataset (json). Random poem or a favorite poem (setting.json)

## Examples

- Tell me a poem
- How about a poem
- Read me a poem
- Recite a poem
- Read poem
- Do you have a favorite poem
