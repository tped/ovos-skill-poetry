# OVOS Poetry Skill

OVOS Skill that reads requested poetry.  Work-in-process

## About

The poetry skill will recite poems from a pre-loaded dataset (json). Options for random poem, a favorite poem or maybe even a poem of the day

## Examples

- Tell me a poem
- How about a poem
- Read me a poem
- Recite a poem
- Read poem
